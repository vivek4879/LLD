# LLD — Low-Level Design Practice

A hands-on repository for learning low-level design, data structures, and object-oriented principles in Java. Each problem is implemented from scratch with a focus on understanding tradeoffs, edge cases, and clean design.

## How to Compile & Run

No build tools — plain Java. From the repo root:

```bash
# Compile all files in a problem directory
javac <ProblemDir>/*.java

# Run the main class
java <ProblemDir>.Main
```

Example:
```bash
javac DynamicArray/*.java
java DynamicArray.Main
```
